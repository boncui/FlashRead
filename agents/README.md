# FlashRead AI Agents

This directory contains AI/LLM agents for FlashRead.

## Getting Started

```bash
# Install dependencies
pnpm install

# Run the example agent
pnpm dev
```

## Structure

- `src/index.ts` - Main entry point and example agent
- Import types and utilities from `@flashread/backend`
